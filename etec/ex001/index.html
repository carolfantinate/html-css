O HTML

O Que é?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas da web. Ele define os elementos que formam o site, como títulos, parágrafos, imagens e links.

Com HTML, é possível criar páginas acessíveis e semânticas, garantindo uma boa experiência para os usuários e permitindo a integração com outras linguagens, como CSS e JavaScript.

Estrutura básica

Um documento HTML começa com a declaração <!DOCTYPE html> e é estruturado com as tags <html>, <head> e <body. No <head>, ficam metadados e links para arquivos externos, enquanto o <body> contém o conteúdo visível da página.

A estrutura do HTML organiza os elementos de forma hierárquica e semântica, facilitando a leitura do código e a exibição correta da página nos navegadores.

Elementos e Atributos

Cada elemento HTML pode ter atributos que modificam seu comportamento. Por exemplo, a tag <img> usa o atributo src para definir a imagem exibida e o alt para acessibilidade.

Os atributos podem ser usados para estilização, acessibilidade e interatividade, tornando o HTML flexível e adaptável para diferentes necessidades.

O CSS

O Que é?

CSS (Cascading Style Sheets) é a linguagem responsável pela estilização das páginas da web. Ele permite definir cores, fontes, layouts e animações, tornando o design mais atraente e responsivo.

Ao separar o conteúdo da apresentação, o CSS facilita a manutenção e atualização do design, garantindo uma experiência visual consistente em diferentes dispositivos.

Seletores e Propriedades

O CSS utiliza seletores para aplicar estilos a elementos HTML. Por exemplo, p { color: blue; } altera a cor do texto dentro de p. Existem seletores específicos, como classes (.classe) e IDs (#id).

Além disso, as propriedades CSS definem características como font-size, background-color e margin, permitindo personalizar a aparência da página conforme necessário.

CSS Responsivo

O design responsivo adapta a aparência do site para diferentes tamanhos de tela. Técnicas como media queries (@media) e unidades flexíveis (em, %, vw) garantem que os elementos se ajustem corretamente.

Com o CSS responsivo, é possível criar layouts fluidos que melhoram a usabilidade em dispositivos móveis e desktops, proporcionando uma experiência consistente para todos os usuários.
